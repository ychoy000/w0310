white-space 속성

normal(기본값)은 연속 공백 들과 줄바꿈이 하나의 공백으로 처리된다. 길면 자동 줄바꿈이 일어난다.텍스트를 입력하면 텍스트 사이의 공백은 하나의 공백으로 처리된다.

nowrap은 연속 공백들과 줄바꿈이 하나의 공백으로 처리된다. 길어도 자동 줄바꿈 없이 박스 밖으로 나간다.

pre는 연속 공백들 과 줄바꿈이 소스 그대로 표현된다. 길어도 자동 줄바꿈이 일어나지 않는다.

pre-wrap은 연속 공백들과 자동 줄바꿈이 일어나며 소스 그대로 표현되나, (강제 줄바꿈-개행, enter,가 된다.) 길면 줄바꿈이 일어난다.

pre-line은 연속 공백들이 하나의 공백으로 처리 된다. 줄바꿈은 소스 그대로 표현되나, 길면 자동 줄바꿈이 일어난다.
