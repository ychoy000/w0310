역안내 페이지 제작

1, HTML, CSS, JS 활용 
  - JS 적용 : 역사 별 슬라이드 이미지 좌우 버튼 클릭 시 좌우 이동
  - 각 역으로 연결 탭 클릭시 역 안내 페이지 변환: 각각의 서브페이지를 제작하여 연결, 공용역은 코레일 연결 링크 그대로 적용(새 탭에서 열림)
2. 슬라이드 이미지: 크기 조정(축소)
3. 역사 상세 정보는 생략
4. 오시는 길: 입력 도안(버스, 지하철, 자가용): iconify에서 받아 .png 파일로 색상 변환하여 사용
5. 주차장 이용안내
  - 주차 공간 안내: 테이블로 제작
  - 주차장 혼잡시간 안내 : 그림 이미지(제작자? 미상) 를 역사 별로 편집 제작하여 사용
